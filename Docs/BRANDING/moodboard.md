# Branding: Moodboard y Estética

Nexora App potencia la identidad visual del proyecto Nexora mediante una estética premium y moderna.

## 🎨 Paleta de Colores Estratégica
| Color | Código Hex | Descripción |
| :--- | :--- | :--- |
| **Primary Blue** | `#0A1F44` | Base robusta que transmite confianza y profesionalismo. |
| **Electric Blue** | `#2563EB` | Acentos tecnológicos y llamadas a la acción (CTAs). |
| **Strategic Purple** | `#6D5EF3` | Innovación, visión estratégica y modernidad. |
| **Action Green** | `#22C55E` | Símbolo de éxito, validación y confirmación. |

## 🔤 Tipografía de Marca
El sistema tipográfico se basa en legibilidad y elegancia:
*   **Sora**: Utilizada principalmente para encabezados y acentos.
*   **Montserrat**: Tipografía secundaria versátil para cuerpos de texto y descripciones.
*   **Outfit**: Fuente complementaria para interfaces limpias y modernas.

## ✨ Estética Visual (Glassmorphism)
La interfaz se construye sobre el concepto de **transparencias y profundidad**:
*   **Glass Cards**: Uso de `rgba(255, 255, 255, 0.05)` con un desenfoque de fondo de `12px` (`backdrop-filter: blur(12px)`).
*   **Bordes Sutiles**: Bordes de `1px` en `white/10` para separar planos visuales sin saturar.
*   **Layout Grid Floor**: Inspiración en estéticas futuristas con estructuras de rejilla claras (1200px máx).

## 🖼️ Logotipo
El logotipo oficial se encuentra integrado en la interfaz (`Logo.png`) con proporciones equilibradas para mantener la coherencia de marca.
*   **Ubicación**: Header persistente y Sidebar de navegación.
*   **Proporción**: Equilibrio entre el isótopo y el texto de marca.
